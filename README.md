# Akuma Tribal Wars

![Akuma Tribal Wars](perfil/imgs/akuma-tw.png)

Repositório para centralizar materiais da conta/tribo Akuma no Tribal Wars: scripts de apoio, modelos de Gerenciador de Construções e imagens de perfil por BR.

## Estrutura

- `src/scripts/`: userscripts e bookmarklets para automações pontuais.
- `src/gc/construções.md`: modelos de Gerenciador de Construções prontos para copiar e colar.
- `perfil/imgs/`: imagens gerais e variações customizadas por mundo.

## Scripts

Os scripts ficam separados por função para facilitar ajuste antes de usar em cada mundo.

- `auto_farm.js`: automação do assistente de farm.
- `coleta.js`: automação de coleta.
- `continuos_recruting.js`: recrutamento contínuo quando a fila está vazia.
- `etiquetador.js`: etiquetas para comandos chegando.
- `fakes.js`: bookmarklet para envio sequencial de fakes.
- `reserva_automatico.js`: reserva aldeias em massa a partir de coordenadas.

## Gerenciador de Construções

Os modelos de GC estão documentados em [src/gc/construções.md](src/gc/construções.md), com população estimada e bloco de código copiável.

## Imagens

A imagem geral do perfil é `perfil/imgs/akuma-tw.png`. As imagens `br139.png`, `br141.png` e `br142.png` são variações customizadas por mundo.
