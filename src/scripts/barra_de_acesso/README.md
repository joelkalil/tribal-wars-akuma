## Como Usar Os Originais

Depois de fazer push para a branch `main`, use as linhas abaixo na barra de acesso do Tribal Wars. Exemplo:

```js
javascript:$.getScript('https://raw.githubusercontent.com/joelkalil/tribal-wars-akuma/refs/heads/main/src/scripts/barra_de_acesso/originals/coinMintingCalculator.js');
```
