# Scripts da Barra de Acesso

Backup dos scripts originais usados na barra de acesso do Tribal Wars.

A pasta `backup/` guarda copias dos arquivos originais. Para usar no jogo, prefira os links oficiais abaixo, porque foram os que funcionaram melhor nos testes.

## Autores Identificados

- Nao informado no arquivo
- RedAlert
- Sophie "Shinko to Kuma"

## Como Usar

Copie a linha do script desejado e cole na barra de acesso do Tribal Wars.

Exemplo:

```js
javascript:$.getScript('https://twscripts.dev/scripts/coinMintingCalculator.js');
```
